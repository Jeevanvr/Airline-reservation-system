# Airline Reservation System

A web-based airline ticket booking and management system developed using PHP and MySQL.

## Features

* User Registration and Login
* Ticket Booking
* Booking Cancellation
* Customer Dashboard
* Database Integration

## Technologies Used

* HTML
* CSS
* JavaScript
* PHP
* MySQL

## Note

The project was originally developed and executed using XAMPP with a MySQL database. The source code is included in this repository.
